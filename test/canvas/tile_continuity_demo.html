Tile Continuity — world-space deterministic seeding

NAIVE — per-tile random seeds
Each tile seeds its own particles → gaps and density jumps at boundaries
WORLD-SPACE — shared deterministic seeds
All tiles share one particle system → seamless crossing at every boundary
The world is 600×600 px. The four tiles each cover a 300×300 quadrant. Left: naive approach — each tile independently picks random particle positions, causing visible density jumps (double-density or gaps) at every boundary crossing. Right: world-space approach — one RainSystem shared by all four tiles; each tile renders only particles whose current position falls inside its viewport. Particles crossing boundaries are seamless because position is a pure function of (particle_id, time) evaluated identically by every tile.
